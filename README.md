## 🚀 Instalación y Uso

### Requisitos
- Navegador web moderno
- Servidor web local (opcional para desarrollo)

### Instalación
1. Clona el repositorio:
```bash
git clone [URL_DEL_REPOSITORIO]
cd portafolio
```

2. Abre `index.html` en tu navegador o usa un servidor local:
```bash
# Con Python 3
python -m http.server 8000

# Con Node.js
npx serve .

# Con PHP
php -S localhost:8000
```

3. Navega a `http://localhost:8000`

## 🌍 Sistema de Idiomas

El portafolio incluye un sistema completo de cambio de idiomas:

- **Español (ES)**: Idioma por defecto
- **Inglés (EN)**: Traducción completa
- **Persistencia**: Las preferencias se guardan en localStorage
- **Cambio dinámico**: Sin recargar la página

### Cómo funciona
```javascript
// Cambiar idioma
changeLanguage('en'); // Cambia a inglés
changeLanguage('es'); // Cambia a español

// Los elementos con atributos data-es y data-en se actualizan automáticamente
```

## 🎨 Personalización

### Colores
Modifica las variables CSS en `src/styles.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #7c3aed;
    --accent-color: #06b6d4;
    /* ... más variables */
}
```

### Contenido
- Edita el HTML para cambiar textos y servicios
- Modifica el objeto `languages` en `src/script.js` para nuevos idiomas
- Ajusta los iconos de Font Awesome según necesites
